# taailsOS
I made my own OS.

## How to run
### Linux (and MacOS)
Open a terminal and type:
```bash
make
chmod +x run.sh
./run.sh
```

> Do the same with the `debug.sh` file for debugging.

### Windows
Open a terminal and type:
```bash
make
./run.sh
```

## Makefiles
In the `Makefile`, you can replace the wcc & wlink path with where you installed them.
