# taailsOS
I made my own os

## How to run
### Linux (and MacOS)
Open a terminal and type:
```bash
make
```
And after that:
```bash
chmod +x run.sh
./run.sh
```

> Do the same with the `debug.sh` file for debugging

### Windows
Open a terminal and type:
```bash
make
./run.sh
```

## Makefiles
In the `Makefile` you can replace the wcc & wlink with your own path
